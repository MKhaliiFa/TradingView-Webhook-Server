TradingView Connection Guide
1
Open an Alert on TradingView
On your chart, click Alerts → Create alert and configure your trigger condition (indicator crossing, price level, etc.).
2
Set the Notification URL
In the Notifications tab, enable Webhook URL and paste your server's webhook endpoint:
https://your-replit-domain/webhook
3
Set the Message Body
In the Message field, paste this JSON:
{% raw %}{ "token": "YOUR_WEBHOOK_SECRET", "action": "{{strategy.order.action}}", "symbol": "{{ticker}}", "price": "{{close}}" }{% endraw %}
4
Alternatively: use the header
Instead of putting the token in the JSON body you can send it as an HTTP header:
X-TV-Token: YOUR_WEBHOOK_SECRET
The header takes priority over the body field when both are present.
Quick cURL Test
curl -X POST https://your-domain/webhook \ -H "Content-Type: application/json" \ -H "X-TV-Token: YOUR_SECRET" \ -d '{"action":"buy","symbol":"BTCUSDT","price":"65000"}'
